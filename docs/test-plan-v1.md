# RepoCheck V1 手动冒烟测试计划

> 无需运行代码，按步骤手动检查即可验证 V1 主流程正确性。

## 前置条件

1. 启动 Docker Compose（MySQL + Redis + MinIO）
2. 启动 AI Service：`cd ai-service && uvicorn app.main:app --port 8123`
3. 启动 Backend：`cd backend && mvn spring-boot:run`
4. 启动 Frontend：`cd frontend && npm run dev`

## 测试步骤

## 已知限制

- V1 中间状态（SEARCHING_REPO/ANALYZING_REPO/GENERATING_REPORT）不可见，因分析在一次 HTTP 调用中完成
- 取消任务不会中断正在执行的 AI 分析（分析完成时检查 CANCELLED 状态并丢弃结果）
- V1 userId 硬编码为 1
